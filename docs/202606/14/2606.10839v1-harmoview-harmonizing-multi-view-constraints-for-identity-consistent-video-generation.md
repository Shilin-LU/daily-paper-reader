## 摘要
当前的 identity-consistent 视频生成方法在大视角变化下难以保持外观保真度。虽然引入多视角参考输入提供了自然的解决方案，但由于缺乏有效的多视角输入框架以及多视角数据的稀缺性，进展仍然受限。针对这些挑战，我们提出了 HarmoView，这是一个鲁棒的身份一致视频生成框架，通过三种架构改进及阶段性训练课程有效整合多视角线索。具体而言，我们首先引入了多级特征注入（Multi-level Feature Injection）以锚定身份保真度；通过跨注意力机制将来自正面参考的原始ViT特征与文本令牌一起注入，MFI提供了持久性的低层外观锚点，补充了DiT模块内的高层身份特征，从而增强了身份保持。然后，我们采用可学习的代理令牌来统一单/多视角设置中的异构参考布局，同时解决参考视角不匹配问题。进一步开发了Jump-RoPE用于身份维度的特征隔离以减少身份串扰。为了激活这些结构能力同时保留原始生成先验，我们提出了渐进视角课程（Progressive View Curriculum）。这种四阶段训练策略采用视角丢弃（view dropout）以促进从普通的T2V生成到高保真、身份持久的空间推理的稳定过渡。此外，我们构建了一个大规模多视角数据集以解决数据稀缺问题。在我们包含52个独特身份、100个手工精选案例的多视角基准上的广泛评估表明，HarmoView显著优于开源基线，并与领先的闭源引擎相匹配，在身份一致视频生成中达到了最先进的性能。

## Abstract
Current identity-consistent video generation methods struggle to preserve appearance fidelity under large viewpoint changes. While introducing multi-view reference input offers a natural solution, progress remains constrained by the lack of effective frameworks for multi-view inputs and the scarcity of multi-view data. We address these challenges by proposing HarmoView, a robust framework for identity-consistent video generation that effectively integrates multi-view cues through three architectural refinements complemented by a staged training curriculum. Specifically, we first introduce Multi-level Feature Injection to anchor identity fidelity; by injecting raw ViT features from frontal references alongside text tokens via cross-attention, MFI provides persistent low-level appearance anchors that complement the high-level identity features within DiT blocks, leading to enhanced identity preservation. Then, we employ learnable proxy tokens to unify heterogeneous reference layouts across single-/multi-view settings while simultaneously resolving the reference-view mismatch problem. Jump-RoPE is further developed for identity-wise feature isolation to reduce identity crosstalk. To activate these structural capabilities while preserving the original generative priors, we propose the Progressive View Curriculum. This four-stage training strategy employs view dropout to facilitate a stable transition from vanilla T2V generation to high-fidelity, identity-persistent spatial reasoning. Furthermore, we construct a large-scale multi-view dataset to address the issue of data scarcity. Extensive evaluation on our multi-view benchmark, comprising 100 manually-curated cases spanning 52 unique identities, demonstrates that HarmoView significantly outperforms open-source baselines and matches leading closed-source engines, achieving state-of-the-art performance in identity-consistent video generation.

## 1. 核心问题与整体含义
- **研究动机**：当前身份一致视频生成方法在大视角变化（如大幅旋转、缩放）时，会出现 **身份漂移**（identity drift）、**几何失真**（geometric distortion）以及“平均脸”效应（mean face），根源在于仅依赖单张正面参考缺乏视角特定的外观证据，导致模型在未见视角上产生幻觉或错误映射。
- **整体含义**：本文提出 **HarmoView**，第一个公开的面向人类的多视角身份一致视频生成框架，通过整合多个视角的参考线索（正面+侧面），有效锚定身份特征，实现高保真的大角度视频合成，并填补了多视角人类视频生成的数据与框架空白。

## 2. 方法论：核心思想与关键技术
### 核心思想
- 在预训练的 Diffusion Transformer（DiT）视频生成主干（Wan2.2-T2V）基础上，将多视角参考图像编码为潜变量并与视频噪声沿序列维度拼接，同时引入三种架构改进和四阶段训练课程，以稳定地激活多视角推理能力。

### 关键技术细节
1. **多级特征注入（Multi-level Feature Injection, MFI）**
   - 从正面参考图像中提取 ViT 视觉 token，通过线性投影后与文本 token 拼接（公式略），构成统一条件序列。
   - 作用：提供低层外观特征作为持久锚点，避免 DiT 高层抽象导致细节丢失，且复用原有 cross-attention 机制，无需新增注意力头。

2. **可学习代理令牌（Learnable Proxy Tokens, LPT）**
   - 对于缺失的参考视角（如仅单视图输入），使用可学习的代理 token 替代，固定序列布局为 N=5 个槽位。
   - 双重作用：
     - **统一布局**：保证训练与推理时序列长度一致，维持 RoPE 位置语义。
     - **注意力沉淀（Attention Sinks）**：吸收不可匹配的注意力权重，缓解参考-视角不匹配问题（如用正面参考生成侧脸时消除错误特征污染）。

3. **Jump-RoPE**
   - 在旋转位置编码（RoPE）中插入索引间隙：
     - **跨身份隔离**：不同身份 token 块之间设置大间隔（如 δ）。
     - **参考与视频解耦**：在参考序列与视频噪声序列之间再插入跳跃。
   - 作用：强制不同身份和不同模态的特征在高维坐标空间中处于分离区域，抑制身份串扰（如将一个人上半脸与另一个人下巴混合）。

4. **渐进式视角课程（Progressive View Curriculum, PVC）**
   - 四阶段训练，逐步增加多视角复杂度：
     - Stage 1：单视图持续训练（利用大量单视图数据，对齐架构修改）。
     - Stage 2：双配置（单视图或完整 N 视图）对齐，学习多视角映射。
     - Stage 3：随机视角丢弃（从 1 到 N 随机采样），迫使模型学习稀疏条件推理。
     - Stage 4：高保真精调（小批量高质量数据，减少丢弃率），优化细节。
   - 作用：避免因多视图数据稀缺导致的过拟合或先验崩溃，保留 T2V 主干生成能力。

5. **可扩展多视角数据管道**
   - 使用 LoRA 微调 FLUX.1 作为视图合成引擎，从正面参考生成 7 个标准视角（0°, ±30°, ±60°, ±90°）。
   - 多阶段过滤：美学预测、模糊检测 → ArcFace 身份相似度（视角自适应阈值）→ 头部姿态估计（误差 ±10°）。
   - 将现有视频逐帧转化为多视角视频-身份对，构建大规模数据集。
