## 摘要
最近的扩散模型支持了在实时流中生成逼真的音频驱动虚拟形象。然而，现有方法难以维持视觉时间一致性，且在复杂的交互式流场景中未能显式感知用户意图。为解决这些挑战，我们提出了InteractiveAvatar，一个实时无限流视频生成框架，支持视觉一致的虚拟形象视频生成和意图感知交互。通过自回归蒸馏，InteractiveAvatar能够实现任意长时间范围内人类虚拟形象的实时流式生成。针对视觉一致性，我们引入了长短时视觉记忆（LSVM）机制，该机制将历史视觉信息灵活压缩为紧凑令牌，同时保持短程连贯性和长程一致性。为了生成与用户意图一致的带有语音和动作的虚拟形象，我们提出了推理-反应模块（RRM），其中包含状态循环策略和缓存切换机制。在多场景下的广泛实验结果表明，我们的方法在长时间生成中达到了最先进的视觉一致性，同时实现了实时复杂的用户-虚拟形象交互。

## Abstract
Recent diffusion-based models have enabled realistic audio-driven avatar generation in real-time streaming. However, existing approaches struggle to maintain visual temporal consistency and fail to explicitly perceive user intent in complex interactive streaming scenarios. To address these challenges, we propose InteractiveAvatar, a real-time infinite-streaming video generation framework that supports visually consistent avatar video generation and intent-aware interactions. With autoregressive distillation, InteractiveAvatar achieves real-time str-eaming generation of human avatars over arbitrarily long durations. For visual consistency, we introduce a Long-Short Visual Memory (LSVM) mechanism that flexibly compresses historical visual information into compact tokens, preserving both short-range coherence and long-term consistency. To generate avatars with speeches and actions aligned with user intent, we propose a Reasoning-Reaction Module (RRM), which incorporates a State-Cycling strategy and a Cache-Switching mechanism. Extensive experimental results over diverse scenarios demonstrate that our method achieves state-of-the-art visual consistency in long-duration generation, while enabling complex user-avatar interaction in real time.

# 交互式虚拟形象：面向一致性及意图感知的实时流式视频生成（InteractiveAvatar）

## 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：现有的基于扩散模型的实时流式音频驱动虚拟形象生成方法，在长时间视频生成中难以维持视觉时间一致性（temporal consistency），且无法显式感知用户意图以生成语义对齐的动作行为（如根据“看手表”指令执行相应动作）。
- **背景需求**：理想的人机交互虚拟形象不仅需要同步唇形与语音，还应能理解用户意图并执行具有上下文关联的动作（如检查手表、拿起物体等），同时保持长时间生成中的外观与画面连贯性。当前方法多局限于简单的“听/说”同步，缺乏对意图的建模和长期一致性的保证。

## 2. 方法论：核心思想、关键技术细节
- **总体框架**：基于 Diffusion Transformer（DiT）和自回归蒸馏，实现实时流式无限时长视频生成。包含三大模块：
  - **自回归蒸馏（Autoregressive Distillation）**：通过 Distribution Matching Distillation（DMD）将双向注意力教师模型蒸馏为少数步的自回归学生生成器，实现实时推理。
  - **长短时视觉记忆（LSVM）机制**：
    - 分为短时记忆（Short-term Memory）和长时记忆（Long-term Memory）。短时记忆保存最近 K 帧的压缩令牌，保证局部连续性；长时记忆存储一组全局代表性令牌（固定容量 N），维持长期全局一致性。
    - 使用轻量压缩网络 C(·) 将视频潜在帧压缩为紧凑令牌。
    - 动态关键帧选择（DKFS）策略：当短时记忆中的帧被弹出时，通过 SigLIP2 提取语义特征，计算长时记忆的全局冗余度评分，选择保留能降低冗余度的新帧，从而自适应更新长时记忆，维护语义多样性。
  - **推理-反应模块（RRM）**：
    - 利用大语言模型（LLM）进行意图理解与状态管理。用户语音经 ASR 转录为文本后输入 LLM，LLM 输出动作状态（action state）和稳定状态（stable state）。动作状态包括动作提示词（pact）和响应音频（aresp）；稳定状态包括稳定提示词（pstable）。
    - **状态循环策略**：生成过程中，模型根据音频驱动唇形同步，根据动作提示指导姿态变化；响应音频结束后切换至稳定提示，保持静态一致性，直至下一轮用户输入。
    - **缓存切换机制**：当动作提示切换时，重新计算前序相邻块的提示条件 KV 缓存，以加速对新提示的适应，降低延迟。
- **训练流程**：四阶段流水线：
  1. 在大规模音视频数据上训练图像-音频到视频（AI2V）模型（双向注意力）。
  2. 预训练记忆压缩模块 C(·) 用于视频重建。
  3. ODE 初始化：使用块因果注意力训练以近似教师 ODE 轨迹。
  4. 自强制 DMD 蒸馏：结合 LSVM 将扩散模型蒸馏为少步自回归生成器。

## 6. 主要结论与发现
- InteractiveAvatar 在对象一致性（OBJ 85.2）、身份保持（ID 4.51）和文本-视频对齐（TV 25.93）上显著优于所有基线，尤其在长时间生成中视觉一致性达到最先进水平。
- 推理速度（FPS 26.68）快于所有对比方法（LiveAvatar 为 21.94），且模型参数量仅 5B，硬件需求更低。
- LSVM 有效缓解了时间漂移，RRM 使虚拟形象能理解并执行用户意图驱动的动作。
- DMD 蒸馏是实现实时性能的关键（无蒸馏时 FPS 仅 1.27）。

## 7. 优点
- **创新性**：首次将长短时视觉记忆与意图推理结合用于实时流式虚拟形象生成，同时解决一致性和交互性问题。
- **技术亮点**：动态关键帧选择（DKFS）基于语义多样性自适应更新长时记忆，避免冗余；状态循环策略实现自然完整的“听-反应-稳定”交互循环；缓存切换机制减少动作切换延迟。
- **实验全面性**：涵盖多种评价指标（质量、一致性、速度），消融实验设计清晰，定性结果直观。
- **性能优异**：在保证高视觉一致性的同时达到实时推理速度（26.68 FPS），且模型轻量（5B）。

## 8. 不足与局限
- **实验的充分性**：测试集仅 500 个视频，且场景为人工设定（特定时间点动作切换），未在真实开放交互对话中评估；消融实验仅报告单次指标值，未提供重复实验的方差，统计显著性不足。
- **潜在偏差风险**：数据集以英语为主，且专有对话数据集未公开，难以复现；模型对 LLM 的依赖可能引入幻觉或意图理解偏差；动作生成依赖于提示文本的语义，对于复杂多步动作（如“先拿书再看时间”）可能执行不精确。
- **应用限制**：当前框架主要针对上肢和头部动作，缺乏全身运动建模；场景演化（如物体出现/消失）依赖预定义的稳定状态提示，无法完全动态；对实时交互中多轮上下文记忆保持仍需进一步验证。
- **资源消耗**：虽相较其他方法更高效，但 64 块 H100 训练仍属高资源消耗，推广门槛较高。
