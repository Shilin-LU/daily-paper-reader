## 摘要
最近，基于Vision Transformer（ViT）的模型在图像超分辨率方面表现出色。然而，ViT的计算复杂度与空间分辨率呈二次方关系，严重限制了其效率，导致高延迟和大量内存消耗。为了缓解这一问题，人们提出了各种基于窗口的注意力机制；然而，这些机制固有地损害了ViT的主要优势——长距离依赖建模。为了克服这些限制，我们提出了聚类单元级相似性Transformer（CUST），一种高效整合全局和局部信息的新型架构。具体来说，CUST使得每个块能够在其局部窗口之外的更广泛区域范围内聚合和关注相似块，从而捕获广泛的上下文理解。此外，它采用重叠注意力窗口来捕获局部依赖关系，同时通过计算原始特征与其下采样-上采样对应特征之间的残差差异来显式提取高频细节。全面的实验表明，我们提出的模型在计算效率和恢复性能之间实现了实际的平衡。在实际约束下，与最近的全局上下文或轻量级模型相比，它实现了更低的内存占用和更快的推理速度。代码可在[https://github.com/jwgdmkj/CUST]获取。

## Abstract
Recently, Vision Transformer (ViT)-based models have exhibited remarkable performance in image super-resolution. However, the quadratic computational complexity of ViTs with respect to spatial resolution severely constrains their efficiency, leading to high latency and massive memory consumption. To alleviate this, various window-based attention mechanisms have been proposed; yet, they inherently compromise the long-range dependency modeling that is the primary advantage of ViTs. To overcome these limitations, we propose the Clustered Unit-level Similarity Transformer (CUST), a novel architecture that efficiently integrates global and local information. Specifically, CUST enables each patch to aggregate and attend to similar patches within a broadened regional scope outside its local window, thereby capturing extensive contextual understanding. Furthermore, it employs overlapping attention windows to capture local dependencies, while explicitly extracting high-frequency details by computing the residual difference between the original features and their downsampled-upsampled counterparts. Comprehensive experiments demonstrate that our proposed model achieves a practical balance between computational efficiency and restoration performance. It achieves a lower memory footprint and faster inference speed compared to recent global context or lightweight models under realistic constraints. Code is available at [https://github.com/jwgdmkj/CUST].

- **问题背景**：Vision Transformer（ViT）在图像超分辨率（SISR）中表现优异，但其自注意力的计算复杂度与输入分辨率呈二次关系，导致高延迟和大量内存消耗。为降低复杂度，窗口化自注意力（WSA）被广泛采用，但它在固定窗口内操作，牺牲了长程依赖建模能力，限制了全局上下文信息的捕获。
- **研究动机**：现有方法（如超令牌生成、大窗口轻量化）存在迭代计算慢、内存占用大或仅优化参数/FLOPs而忽略实际硬件指标（延迟、峰值内存）等瓶颈。因此，需要一种既能高效集成全局与局部信息、又在实际部署中兼顾计算效率与重建质量的轻量级超分架构。
- **整体含义**：论文提出的CUST（Clustered Unit-level Similarity Transformer）通过聚类相似性单元，在扩大搜索区域内聚合相似patch，同时利用多频误差信号显式恢复高频细节，实现了优良的效率-性能平衡。

## 2. 方法论

### 核心思想
- 整体架构由多个深度特征提取块组成，每个块包含**跨窗口亲和邻居注意力（CANA）** 模块和**多频误差驱动密集注意力（MEDA）** 模块，辅以卷积前馈网络（ConvFFN）和层归一化。

### 关键技术细节
- **CANA（跨窗口亲和邻居注意力）**：
  1. 将特征图划分为多个搜索区域（grid size `GS`，为窗口大小`ws`的倍数），在每个区域内通过平均池化生成代表性窗口令牌（每个窗口对应一个令牌）。
  2. 计算区域内每个patch与所有窗口令牌的点积相似度，通过Argmax将每个patch分配到最相似的窗口令牌索引，形成语义簇。
  3. 为避免簇边界信息断裂，将当前簇的查询与相邻簇（`C_{n-1}`, `C_{n+1}`）的各一半（最近语义部分）拼接，作为扩展的Key/Value。
  4. 应用注意力掩码（仅允许相同窗口ID的patch互相作用），并通过可学习门控（Gate）进行自适应调制。
- **MEDA（多频误差驱动密集注意力）**：
  1. 对输入特征`X`进行下采样再上采样得到`X_low`，计算残差`X_error = X - X_low`（高频误差）。
  2. 通过1×1卷积和空洞卷积（Dilated Conv）对`X_error`进行细化，再经GELU激活和1×1卷积生成空间门控图`G`。
  3. 将细化后的误差与门控逐元素相乘，乘以可学习标量`α`后加回原特征，进行信息补偿。
  4. 最后应用重叠窗口自注意力（Overlap Self-Attention）进一步细化局部细节。

### 算法流程（文字说明）
1. 输入特征→CANA：聚类、扩展Key/Value、掩码注意力、门控→输出全局增强特征。
2. 输出特征→MEDA：多频误差提取、误差细化与门控、特征补偿、重叠窗口注意力→输出局部细化特征。
