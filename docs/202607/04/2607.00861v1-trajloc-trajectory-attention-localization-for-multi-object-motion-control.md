## 摘要
控制图像到视频（I2V）生成中多个对象的运动，需要在遵守不同目标轨迹的同时保持对象身份。随着对象数量增加以及路径相交或相互遮挡，这一任务变得尤为具有挑战性。现有方法将多条轨迹纠缠在共享的密集条件信号中，使得在拥挤场景中难以保持对象级对应性。我们摆脱了这种范式，对每个对象施加严格的独立空间约束。我们的方法TrajLoc直接在注意力层中实现这一点，通过将每个对象令牌的交叉注意力权重替换为以其每帧目标位置为中心的高斯热图。相同的每个对象令牌接口通过学习的嵌入传递轨迹和深度，并通过编码第一帧外观来保持身份，以替代对象令牌。在六个数据集上的评估表明，我们的方法在视觉保真度和轨迹遵守方面一致提升，这些数据集包含最多20个同时控制的对象以及分布外的真实世界场景。应用于两个架构不同的主干网络（CogVideoX 5B 和 WaN 2.1 14B），与最强基线相比，我们的方法平均PSNR提高了+4.3 dB，轨迹终点误差降低了51%。项目页面：https://sela-omer.github.io/traj-loc/

## Abstract
Controlling the motion of multiple objects in image-to-video (I2V) generation requires preserving object identities while enforcing adherence to distinct target trajectories. This becomes particularly challenging as the number of objects increases and their paths intersect or occlude one another. Existing approaches entangle multiple trajectories within a shared, dense conditioning signal, making object-level correspondence difficult to preserve in crowded scenes. We depart from this paradigm and enforce a strict, per object spatial constraint that isolates instances independently. Our method, TrajLoc, achieves this directly within the attention layers by substituting the cross-attention weights of each object token with a Gaussian heatmap centered on its target location at every frame. The same per object token interface carries trajectory and depth through a learned embedding and preserves identity by encoding first frame appearance in place of an object token. Evaluations across six datasets, featuring up to 20 simultaneously controlled objects and out of distribution real world scenes, demonstrate that our method consistently improves both visual fidelity and trajectory adherence. Applied to two architecturally distinct backbones (CogVideoX 5B and WaN 2.1 14B), our approach achieves average gains of +4.3 dB PSNR and a 51% reduction in trajectory end point error compared to the strongest baselines.
